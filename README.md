# Discord Server Bot for My Roblox Game

This bot is designed to integrate with a Roblox game server, providing logs, automation, and moderation features. It helps manage game events, player activity, and server actions in real time.

## Features
- Player join/leave logs
- Command handling
- Automated moderation
- Customizable for your own Roblox game
